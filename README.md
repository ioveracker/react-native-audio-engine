```

```

## Contributing

### Set Up

```
yarn
```

### Build

```
yarn build
```

### Run Example

```
cd example
yarn
```

Then either run on iOS:

```
yarn ios
```

or Android:

```
yarn android
```

### Update Example With NativeModule Changes

After making changes to the NativeModule, be sure to clean the example and reinstall dependencies:

```
cd example && yarn clean && yarn && yarn ios
```
